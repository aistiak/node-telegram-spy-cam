## SpyCam 

- install dependencies `yarn install`
- create .env file 
```
TELEGRAM_TOKEN=your-bot-token
MY_TELEGRAM_CHAT_ID=your-chat-id
ACTIVATION_TEXT=snap
```
- run main.js `node main.js`
